# 前端工程師履歷 | Frontend Engineer Resume

一個使用 Vue.js 3 + Element Plus + Tailwind CSS 構建的履歷網站。

A modern, responsive resume website built with Vue.js 3, Element Plus, and Tailwind CSS.

## 🛠️ 技術棧 | Tech Stack

### 前端框架
- **Vue.js 3** - 漸進式 JavaScript 框架
- **Vite** - 下一代前端構建工具

### UI 組件庫
- **Element Plus** - Vue 3 的桌面端組件庫
- **@element-plus/icons-vue** - Element Plus 圖標庫

### 樣式方案
- **Tailwind CSS 4** - 實用程式優先的 CSS 框架
- **PostCSS** - CSS 後處理器
- **Autoprefixer** - 自動添加瀏覽器前綴


### 組件結構
```
src/
├── App.vue          # 主要組件
├── main.js          # 應用入口
└── style.css        # 全局樣式和 CSS 變量
```

## 🎨 設計特色 | Design Features

- **漸變背景** - 從深藍到淺藍的漸變背景
- **玻璃質感** - 毛玻璃效果的卡片設計
- **圓角設計** - 現代化的圓角元素
- **顏色系統** - 紫色主題色搭配漸變效果
- **響應式布局** - 適配各種螢幕尺寸

## 📦 項目結構 | Project Structure

```
Resume/
├── public/              # 靜態資源
├── src/
│   ├── App.vue         # 主組件
│   ├── main.js         # 應用入口
│   └── style.css       # 全局樣式
├── index.html          # HTML 模板
├── package.json        # 依賴配置
├── vite.config.js      # Vite 配置

└── postcss.config.js   # PostCSS 配置
```
