# Frontend Engineer Resume

A modern, responsive resume website built with Vue.js for Apple Frontend Engineers.

## Features

- 🎨 Modern, Apple-inspired design
- 📱 Fully responsive layout
- ✨ Smooth animations and transitions
- 🚀 Fast performance with Vite
- 🌐 Easy deployment to GitHub Pages

## Development

### Prerequisites

- Node.js 16+
- npm or yarn

### Setup

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

### Customization

1. Update personal information in `src/App.vue`
2. Modify styling in `src/style.css`
3. Add your photo by replacing the avatar placeholder
4. Update contact information and social links

### Deployment

The project is configured for automatic deployment to GitHub Pages. Simply:

1. Push to the `main` branch
2. GitHub Actions will automatically build and deploy
3. Access your resume at `https://yourusername.github.io/resume/`

## Tech Stack

- Vue.js 3
- Vite
- CSS3 with modern features
- GitHub Pages deployment
